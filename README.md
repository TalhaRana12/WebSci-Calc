# WebSci Calc üßÆ

A robust scientific calculator for the web that bridges the gap between traditional handheld calculators and modern web interfaces. This project supports standard arithmetic as well as complex scientific functions including trigonometry, logarithms, and exponential values.

## üöÄ Features

*   **Scientific Functions:** Square ($x^2$), Cube ($x^3$), Square Root ($\sqrt{}$), $\sin$, $\cos$, $\tan$, $\log_{10}$, and $\ln$.
*   **Constants:** Built-in support for $\pi$ and Euler's number ($e$).
*   **Dual Input:** Fully functional via mouse clicks or keyboard typing.
*   **Keyboard Accessibility:** Custom key mappings for scientific functions (e.g., press 's' for sin).
*   **Responsive Design:** Works on desktop and mobile devices.
